<h1>Employee Shift Management System</h1> <p> The Employee Shift Management System is a web-based application developed to simplify and automate employee shift scheduling, leave management, and attendance tracking within an organization. This project reduces the complexity of manual workforce management by providing an efficient and centralized platform for administrators, managers, and employees. The system is built using a REST API-based architecture with React for the frontend, Spring Boot for the backend, and MySQL for database management. </p> <h2>Table of Contents</h2> <ol> <li>Project Overview</li> <li>Technologies Used</li> <li>Modules and Functionalities</li> <li>System Architecture</li> <li>Team Members</li> <li>Conclusion</li> </ol> <h2>Project Overview</h2> <p> The traditional method of managing employee shifts manually can lead to scheduling conflicts, inaccurate attendance records, and inefficient leave tracking. The Employee Shift Management System was developed to overcome these challenges by automating scheduling and employee management processes. </p> <p> This system provides an organized platform where administrators and managers can efficiently manage employee details, assign shifts, approve leave requests, and monitor attendance records in real time. </p> <h3>Core Features</h3> <ul> <li>Employee Management</li> <li>Shift Scheduling</li> <li>Leave Management</li> <li>Attendance Tracking</li> <li>Notifications and Alerts</li> </ul> <h3>Architecture</h3> <p> The project follows a REST API-based architecture where the frontend and backend communicate through HTTP requests. The frontend is developed using React.js to provide a responsive and user-friendly interface, while the backend is built using Spring Boot to handle business logic and API services. MySQL is used for secure and efficient data storage. </p> <h2>Technologies Used</h2> <ul> <li><b>Frontend:</b> React.js</li> <li><b>Backend:</b> Spring Boot (Java)</li> <li><b>Database:</b> MySQL</li> <li><b>ORM Framework:</b> JPA & Hibernate</li> <li><b>Security:</b> Spring Security</li> <li><b>API Testing:</b> Postman</li> <li><b>Version Control:</b> Git & GitHub</li> <li><b>Development Environment:</b> Spring Tool Suite (STS), VS Code</li> </ul> <h2>Modules and Functionalities</h2> <h3>1. Employee Management</h3> <ul> <li>Creating, updating, and managing employee profiles.</li> <li>Assigning employee roles such as Admin, Manager, and Employee.</li> <li>Managing department information and employee records.</li> <li>Performing CRUD operations for employee details.</li> </ul> <h3>2. Shift Scheduling</h3> <ul> <li>Creating and modifying employee shift schedules.</li> <li>Preventing overlapping shift assignments.</li> <li>Viewing daily, weekly, and monthly shift schedules.</li> <li>Managing real-time shift updates using batch processing.</li> </ul> <h3>3. Leave Management</h3> <ul> <li>Employees can apply for leave requests online.</li> <li>Managers can approve or reject leave applications.</li> <li>Tracking leave balances and leave history.</li> <li>Managing different leave types.</li> </ul> <h3>4. Attendance Tracking</h3> <ul> <li>Employee check-in and check-out functionality.</li> <li>Automatic calculation of working hours.</li> <li>Generating attendance reports.</li> <li>Monitoring employee attendance records efficiently.</li> </ul> <h3>5. Notifications and Alerts</h3> <ul> <li>Sending notifications for upcoming shifts.</li> <li>Alerts for pending leave approvals.</li> <li>Notifications regarding leave request status.</li> <li>Providing important system updates to employees and managers.</li> </ul> <h2>System Architecture</h2> <ul> <li><b>Frontend:</b> React.js user interface for employee and admin interactions.</li> <li><b>Backend:</b> Spring Boot REST APIs for business logic and data handling.</li> <li><b>Database:</b> MySQL database for storing employee, shift, leave, and attendance records.</li> <li><b>Authentication & Authorization:</b> Spring Security for secure access control.</li> </ul> <h2>Project Developer</h2>
