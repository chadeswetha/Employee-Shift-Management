<h1>Employee-Shift-Management-System</h1>

<p>The Employee Shift Management System is designed to simplify the process of scheduling employee work shifts, managing leaves, and tracking attendance. It features a REST API-based backend architecture and uses React for the frontend. The system is developed using Java Spring Boot framework. Key modules include Employee Management, Shift Scheduling, Leave Management, Attendance Tracking, and Notifications & Alerts, ensuring efficient workforce management.</p>

<h2>Model Overview</h2>

<p>
The Employee Shift Management System helps organizations automate employee scheduling and attendance management processes. The system reduces manual workload, minimizes scheduling conflicts, and improves workforce efficiency.
</p>

<ol>
  <li>Brief introduction to the problem of manual shift scheduling.</li>

  <li>Overview of the system's core functionalities:
    <ol>
      <li>Employee Management</li>
      <li>Shift Scheduling</li>
      <li>Leave Management</li>
      <li>Attendance Tracking</li>
      <li>Notifications and Alerts</li>
    </ol>
  </li>

  <li>REST API-based architecture for frontend and backend communication.</li>

<h2>Technologies Used</h2>

<ul>
  <li>Frontend: React.js</li>

  <li>ORM Framework: JPA & Hibernate</li>
